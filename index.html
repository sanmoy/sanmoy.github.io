Classic Triangle Puzzle Game

Overview

A classic puzzle for everyone. Gameplay is colorful, intuitive and challenging.

An age old Chinese puzzle game and unique of its kind. This game is for all age group, 8 to 80.

Gameplay is simple. 12 colorful pieces to put together inside a triangle. Player can rotate an element, flip an element and move it to its correct position. Some elements are symmetric. They cannot be rotated. Some are mirror asymmetric. Only these elements can be flipped.

Play the tutorial to know the basic moves. Then you can dive into the game. There are total 350 stages. Enough to keep you thinking for months.

This is an addictive logical puzzle. Parents can play with their kids as an educational game which will help kids to learn how different objects can be put together.
